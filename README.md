# dfind - redundant data find

dfind is a command line tool that finds duplicate files.

## Installation

```sh
go run dfind.go
go build dfind.go
```

or

```sh
make
```

## Usage

```sh
dfind [directory1] [directory2] [directory3]...
```

If you dont specify any directory, it will use current directory.
